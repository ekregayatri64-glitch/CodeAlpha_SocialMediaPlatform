# CodeAlpha Social Media Platform

## Project Description

ConnectHub is a mini social media platform developed using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose. Users can register, log in, create posts, like posts, comment on posts, and follow profiles.

## Features

* User Registration
* User Login
* User Profile Page
* Create Posts
* Like Posts
* Comment on Posts
* Follow System
* MongoDB Database Integration
* Responsive User Interface

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Installation

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the server:
   node server.js
4. Open the project in your browser
